# Open Questions

**Purpose:** founder-only input queue  
**Rule:** questions appear here only when Claude or Codex cannot resolve them
within current ownership rules.

## Usage

For each question:

```md
## Q-YYYYMMDD-N — Short title

- **Asked by:** Claude | Codex
- **Why founder input is required:** one sentence
- **Decision needed:** one sentence
- **Recommended default if no answer by deadline:** one sentence
- **Blocks:** what work is blocked, if anything
```

Once answered, move the resolution into `decision_log.md` and remove the open
question from this file.
